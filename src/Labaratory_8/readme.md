## **Лабараторная работа №8**
***
Создать **три класса**. 
- **Первый** описывает человека. У него усть поля: _Имя, Фамилия, год рождения_.
- **Второй** описывает автомобиль. У него есть поля: _Марка, Год выпуска, Объем двигателя_.
- **Третий** описывает книгу. У него есть поля: _Название, Автор, Год выпуска_.

В каждом классе **создать конструктор и метод**, который выводит значения всех полей.
В одном классе **создать метод**, который принимает в качестве параметра два других объекта и выводит строку в формате:

_<Имя человека> читает книгу <Название книги> про машину <Название машины>_
