## **Лабараторная работа №11**
***
Создать класс Person с private полем возраст и public полями ФИО, Телефон.  
Создать **класс-наследник Student** с дополнительными **public** полями - _Курс, Факультет_ и private полем _дата поступления_.

1. ФИО и возраст всех студентов 
2. список студентов заданного факультета; 
3. список студентов, поступивших после заданного года.
