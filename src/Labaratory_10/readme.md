## **Лабараторная работа №10**
***
Создать класс **Animal** и расширяющие его классы **Dog, Cat, Horse**.

Класс **Animal** содержит переменные _food, location_ и методы _makeNoise, eat, sleep._ 
Метод **_makeNoise_**, например, может выводить на консоль "Такое-то животное спит".  
_Dog, Cat, Horse_ переопределяют методы makeNoise, eat.  
Добавьте переменные в классы **Dog, Cat, Horse**, характеризующие только этих животных.  
Создайте класс **Ветеринар**, в котором определите метод _void treatAnimal(Animal animal)_. 
Пусть этот метод распечатывает **food** и **location** пришедшего на прием животного.
